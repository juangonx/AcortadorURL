# Acortador de Magnet URI


Proyecto web minimalista que consume la API abierta de **mgnet.me** para acortar Magnet URI.


## API utilizada
- Endpoint: http://mgnet.me/api/create
- Método: GET
- Parámetros:
- m: Magnet URI
- format: json


## Tecnologías
- HTML5
- CSS3
- JavaScript (Fetch API)


## Uso
1. Abrir `index.html`
2. Pegar un Magnet URI
3. Presionar **Acortar**
4. Obtener la URL corta


## Notas
- Las carpetas `images` e `icons` están listas para futuros recursos
- Proyecto funcional sin dependencias externas
